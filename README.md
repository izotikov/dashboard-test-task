# Тестовое задание плиточный интерфейс

Стек - React + Typescript + Vite + VanillaCSS, архитектурный подход FSD v2.1

## Инструкция по запуску проекта

1. Локально клонировать репозиторий
2. yarn install
3. yarn dev
